# QA
# Installation

  !Require MySQL

```
    > git clone https://github.com/mirchenko/qa.git yourfolder
    > cd yourfolder
    > npm install
    > Execute sql file -> qa.sql
    > cd server
    > Set up your db config in file squelize.js
```

# Usage

```
    > npm run server
    > npm start
```
Open your browser and navigate to [http://localhost:8080](http://localhost:8080)
